# readme-social-icons

These images can be used as links in READMEs, but you can also freely download and use any of them. I don't claim ownership over any of the images, although I tweaked a lot on the SVGs in the "filled" folder.

To use them as links, enter the following url, while replacing "size/example_folder/example.svg":
```
https://raw.githubusercontent.com/CLorant/readme-social-icons/main/size/example_folder/example.svg
```

There are duplicate images in some folders, just to ensure that there aren't any "not found" errors.
